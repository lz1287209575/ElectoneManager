## 🎵 MIDI 文件详细结构

### 📄 D-pipayu 02 (演奏版)
**文件**: `SONG_001/MDR_000.MID` (55 KB)

**基本参数**:
```
格式        : MIDI Format 0 (单轨道，多通道)
分辨率      : 960 ticks/beat (高精度，标准工作站级)
总音符数    : 3,137 个音符
SysEx消息   : 19 条 (Yamaha XG 配置命令)
```

**速度与拍号**:
```
第 0 beat  : 120 BPM  [4/4 时间签名]
第 4 bar   : 102 BPM  [速度渐慢]
```

**16 个 MIDI 通道音色分配**:

| Ch | ELS-02C 角色 | 声音名称 | 音符数 | 用途 |
|:--:|:------------|:--------|:-----:|:-----|
| **0** | 上排键盘 (UKbd) | SysEx/XG定制 | 272 | **主旋律** |
| **1** | 下排键盘 (LKbd) | SysEx/XG定制 | 334 | **辅助旋律** |
| **2** | 脚踏板 (Pedalboard) | SysEx/XG定制 | 89 | **低音支撑** |
| **3** | 主导音 (Lead) | (无) | 0 | 预留 |
| **4** | 键盘打击乐 (KBP) | 钟琴 #112 | 63 | **敲击声** |
| **5** | 自动贝司 Ch1 | 尼龙吉他 #24 | 276 | **贝司线** |
| **6** | 自动贝司 Ch2 | 弦乐合奏1 #48 | 113 | **和声衬底** |
| **7** | 节奏 Ch1 | 弦乐合奏2 #49 | 144 | **伴奏** |
| **8** | 节奏 Ch2 | 尺八 #77 | 36 | **笛音效果** |
| **9** | 节奏 Ch3 (鼓) | 合唱 #52 | 29 | **打击节奏** |
| **10** | XG Part 1 | 扬琴 #15 | **590** | **独奏乐器 🎵** |
| **11** | XG Part 2 | 西塔琴 #104 | **590** | **独奏乐器 🎵** |
| **12** | XG Part 3 | 合成垫音 #91 | 29 | **背景垫音** |
| **13** | XG Part 4 | 竖琴 #46 | **572** | **琶音效果 🎵** |

| **15** | 控制/速度 | 钢琴 #0 | 0 | 系统控制 |

**主要特点**:
- ✅ **三层组织**: 键盘部分(Ch0-2) + 伴奏部分(Ch5-9) + 效果部分(Ch10-13)
- ✅ **音色设计**: 扬琴(590音) + 西塔琴(590音) + 竖琴(572音) = 1,752 音符的独奏层
- ✅ **SysEx控制**: 通过 Yamaha XG 消息精细配置每个通道的混响、合唱等效果

## ⚙️ 二进制音色文件 (REG_001.B00)

### 📦 SysEx 批量转储结构
```
Size       : 270,095 字节
Format     : MIDI SysEx (F0 43 70 78 ...)
Yamaha ID  : 0x43 (制造商代码)
Device ID  : 0x70 (XG格式标识)
Msg Type   : 0x78 (批量数据转储)
Address    : 0x013C00 (XG存储器起始地址)
```

### 📊 字节分布特征

```
0x00 (38.5%) ████████████████████████ 零填充/未使用区域
0x40 (12.4%) ████████ 数据值
0xFF (7.6%)  █████ 哨兵/最大值标记
0x7F (6.8%)  █████ 出现 18,386 次
0x01 (2.2%)  █ 控制字节

其他 (32.5%) ██████████████████ 混合参数
```

### 📋 包含的配置数据

```
✓ 14 通道的音色/音块 (Voice Assignment)
✓ 混响、合唱、延迟等效果参数 (Reverb/Chorus/Delay)
✓ 力度曲线和表达式范围 (Velocity Curve, Expression Range)
✓ 打击乐音色映射 (Drum Kit Mapping)
✓ 包络生成器设置 (ADSR Envelope)
✓ LFO 和调制参数 (Modulation)
✓ 琶音器和其他效果器设置
```

## 🎯 适用于 ElectoneManager 的设计参考

### 1️⃣ 多版本支持
- **模式识别**: 通过MIDI中键盘通道(Ch0-2)是否包含音符来识别"演奏"vs"伴奏"模式
- **UI呈现**: 显示可选的"仅伴奏"开关

### 2️⃣ 16通道路由设计
```
通道0-2   → 键盘输入 (可选)
通道3     → 预留/主导
通道4     → 打击乐 
通道5-9   → 自动伴奏 (可选，由律动节奏选择)
通道10-13 → 配置音色 (由预设控制)
通道15    → 系统控制/速度
```

### 3️⃣ 预设存储格式
- REG 文件 (270KB SysEx 转储) = **完整音色快照**
- 包含所有 XG 参数、混响、合唱等全局设置
- 建议压缩存储或拆分为多个小文件

### 4️⃣ MIDI 播放引擎建议
```
1. 加载 REG 文件 → 初始化14个通道的音色/效果
2. 播放 MDR MIDI → 960 ticks/beat 分辨率支持
3. 响应 SysEx  → 运行时通道参数调整
4. 显示进度 → 根据活跃通道动画更新 UI
```
